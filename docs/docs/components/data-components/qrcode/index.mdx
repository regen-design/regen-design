# 二维码 QR Code

二维码是一种矩阵式二维条码，是用某种特定的几何图形按一定规律在平面（二维方向上）分布的黑白相间的图形记录数据符号信息的。主要用于快速识别数据、信息的自动识别、记录、存储等。
