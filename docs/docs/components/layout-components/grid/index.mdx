# 栅格 Grid

Grid 响应式布局网格可适应屏幕尺寸和方向，确保布局之间的一致性。

## 演示

<BrowserOnly>
  {() => (<ThemeProvider>
    <Grid yGap={16} cols={1}>
      <CodeBlock title="基础"
                 code={String(BasicCode)}>
        <Basic />
      </CodeBlock>

      <CodeBlock title="间隔"
                 code={String(GapCode)}>
        <Gap />
      </CodeBlock>

      <CodeBlock title="偏移"
                 code={String(OffsetCode)}>
        <Offset />
      </CodeBlock>

      <CodeBlock title="示例样式"
                 onlyCode
                 code={String(DemoCss)}>
      </CodeBlock>
    </Grid>
  </ThemeProvider>)}
</BrowserOnly>
