# 间距 Space

Space 组件管理其直接子级沿垂直或水平轴的布局，每个子级之间具有可选的间距和分隔线。

## 演示

               code={String(BasicCode)}>

    </CodeBlock>
  </Grid>
</ThemeProvider>
