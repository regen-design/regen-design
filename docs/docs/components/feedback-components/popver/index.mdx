# 弹出信息 Popover

Popover 在内容周围弹出一些隐藏的信息。Popover 里面没什么内置样式，在里面填什么主要靠你了。

## 演示

<BrowserOnly>
  {() => (<ThemeProvider>
    <Grid xGap={16} cols={2} style={{
      alignItems: 'flex-start'
    }}>
      <Grid yGap={16} cols={1}>
        <CodeBlock title="基础"
                   code={String(BasicCode)}>
          <Basic />
        </CodeBlock>
       <CodeBlock title="位置"
                   code={String(PositionCode)}>
          <Position />
        </CodeBlock>
        <CodeBlock title="不要基础样式"
                   code={String(CustomCode)}>
          <Custom />
        </CodeBlock>
      </Grid>
      <Grid yGap={16} cols={1}>
       <CodeBlock title="触发条件"
                   code={String(TriggerCode)}>
          <Trigger />
        </CodeBlock>
        <CodeBlock title="不用箭头"
                   code={String(NoArrowCode)}>
          <NoArrow />
        </CodeBlock>
      </Grid>
    </Grid>
  </ThemeProvider>)}
</BrowserOnly>
