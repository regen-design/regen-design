# 动态标签 Dynamic Tags

把标签变得可以输入。

## 演示

<BrowserOnly>
  {() => (<ThemeProvider>
    <Grid xGap={16} cols={2} style={{
      alignItems: 'flex-start'
    }}>
      <Grid yGap={16} cols={1}>
        <CodeBlock title="基础"
                   code={String(BasicCode)}>
          <Basic />
        </CodeBlock>
      </Grid>
      <Grid yGap={16} cols={1}>
        <CodeBlock title="最大标签数量"
                   code={String(MaxCode)}>
          <Max />
        </CodeBlock>
      </Grid>
    </Grid>
  </ThemeProvider>)}
</BrowserOnly>
