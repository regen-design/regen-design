# 图标 Icons

内部使用 <a href="https://www.flaticon.com" target="_blank">flatIcon</a> 作为图标库。

## 使用方法

使用图标组件，你需要安装 <a href="https://www.npmjs.com/package/@regen-design/icons" target="_blank">@regen-design/icons</a> 图标组件包

<Tabs>
  <TabItem value="npm" label="npm" default>
    ```shell
    npm install @regen-design/icons --save
    ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
    ```shell
    yarn add @regen-design/icons --save
    ```
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    ```shell
    pnpm add @regen-design/icons --save
    ```
  </TabItem>
</Tabs>
